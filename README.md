#  Superstore Sales Analysis using Python, MySQL & Power BI

##  Project Overview
This project analyzes Superstore sales data to extract meaningful business insights related to sales performance, profitability, customer behavior, and regional trends.  
An end-to-end data analytics workflow is implemented using **Python**, **MySQL**, and **Power BI**.

---

##  Project Objectives
- Analyze overall sales and profit trends
- Identify top-performing regions, categories, and sub-categories
- Understand customer purchasing behavior
- Create an interactive Power BI dashboard for business decision-making

##  Dataset Information
- **Dataset:** Superstore Sales Data
- **File:** `super_store_dataset.csv`
- **Description:**  
  Contains order-level details including sales, profit, category, sub-category, region, customer information, and order dates.

---

##  Project Workflow
1. **Data Cleaning (Python)**
   - Removed missing and duplicate values
   - Converted data types
   - Prepared structured data for analysis

2. **Data Analysis (MySQL)**
   - Sales and profit aggregation
   - Region-wise and category-wise analysis
   - Time-based trend analysis

3. **Visualization (Power BI)**
   - Sales and Profit KPIs
   - Category and Sub-category breakdown
   - Region-wise performance
   - Monthly and yearly trends
   - Interactive slicers and filters


---

##  Key Insights
- Technology category contributes the highest revenue
- Certain regions generate higher profit margins
- Sales show clear monthly and yearly trends
- Repeat customers have higher purchasing value
